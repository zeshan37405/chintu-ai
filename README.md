# Chintu AI

Chintu AI is an Urdu-first Android mobile assistant focused on reliable local command execution on Redmi Note 11 / Android 13 / HyperOS.

## Stable 3.0 features

- Speech-recognition state machine with start, listening and result watchdogs
- Redmi/HyperOS busy-service recovery, bounded retry and Google system-speech fallback
- Urdu/English command routing with typed-command fallback
- Contact lookup, number display, dialer and SMS by saved name
- Installed-app launching by aliases or visible launcher label with ambiguity selection
- Google/YouTube/weather/maps search, alarms, timers, torch, camera and settings
- Local command history and defensive intent/error handling
- No API key or secret embedded in the app

Financial transactions, password changes and other sensitive account actions are intentionally blocked.
